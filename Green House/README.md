# Green House

Green House Alarm

This project aims to monitor the neccessary condition of Green House; if it fails anytime then alarm is blown.

### Components:

Name|Quantity
----|--------
Temperature | 1
Humidity | 1
Brightness | 1
Push Button | 1
RGB LED  | 1
Speaker | 1
Breadware Board | 1
USB cable | 1

### Connection: 
```
Temperature: M5
Humidity: M6
Brightness: M7
RGB LED: M10
Push Button: M11
Speaker: M9
```
